# Concrete Kingdom

An open-world crime sandbox game. One district. Your empire.

Built with **Three.js** (web prototype) and **Godot 4** (native port). Inspired by GTA V.

## Quick Start (Web)

```bash
cd concrete-kingdom
npm install
npx vite
```

Open http://localhost:5173 in your browser.

## Quick Start (Godot 4)

Open `concrete-kingdom-godot/project.godot` in Godot 4.7+.

## Controls

| Key | Action |
|---|---|
| WASD | Move / Drive |
| Shift | Sprint |
| Space | Jump |
| E | Interact / Enter/Exit Vehicle / Takeover Business |
| R | Rob Business |
| Q | Vault |
| Mouse | Look around |
| Tab | Lock/unlock mouse |

## Features
- Third-person open-world gameplay
- 9 buildings including a bank with enterable interior
- Player vehicle (Porsche 911 Carrera 4S)
- Police AI with 5-star wanted system
- Business takeover and robbery mechanics
- Currency system with passive income
- Day/night cycle
- Mission system (5 missions, branching objectives)
- Faction system (Mafia, Syndicate, Police)

## Tech Stack
- **Web:** Three.js + Vite
- **Native:** Godot 4 + GDScript
